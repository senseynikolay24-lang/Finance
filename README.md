# Финансы — приложение для отслеживания финансового здоровья

Личное приложение на русском языке для контроля финансов: учёт доходов и расходов,
отслеживание закрытия долгов, кредитов и ипотеки, финансовые цели, вклады и
инвестиции. Работает офлайн, все данные хранятся локально на устройстве (IndexedDB)
и никуда не отправляются.

Приложение является PWA — его можно установить на телефон как обычное приложение
и открывать с домашнего экрана.

## Возможности

- **Счета и карты** — карты, наличные, счета с автоматическим пересчётом баланса.
- **Операции** — доходы, расходы, переводы между счетами, снятие наличных,
  пополнение карты. Категории и подкатегории с иконками.
- **Кредиты и ипотека** — аннуитетный и дифференцированный платёж, полный график
  амортизации, остаток долга, переплата, прогресс закрытия. Кредитные карты с
  лимитом и грейс-периодом. История платежей.
- **Цели** — накопительные цели с прогресс-баром, сроком и пополнением.
- **Вклады** — депозиты со ставкой, сроком и капитализацией (в т.ч. ежемесячной);
  прогноз дохода к концу срока.
- **Инвестиции** — брокерский счёт и ИИС, позиции по акциям, ручное обновление
  котировок, стоимость портфеля и прибыль/убыток (P/L).
- **Статистика** — периоды День / Неделя / Месяц / Год, график динамики,
  разбивка по категориям (пончик) с детализацией по подкатегориям (drill-down).
- **Бюджеты (план/факт)** — лимиты по категориям на месяц и контроль превышения.
- **Резервное копирование** — экспорт и импорт всех данных в JSON.

## Технологии

React 18 + TypeScript, Vite, Dexie (IndexedDB), React Router, Zustand, Recharts,
date-fns, Tailwind CSS, vite-plugin-pwa.

## Запуск

```bash
npm install
npm run dev      # разработка (http://localhost:5173)
npm run build    # production-сборка (PWA)
npm run preview  # предпросмотр собранной версии
npm run test     # юнит-тесты финансовой логики (Vitest)
```

## Структура

```
src/
  db/          — модель данных Dexie (types, db, seed) и репозиторий (repo.ts)
  lib/         — finance.ts (расчёты), analytics.ts (аналитика),
                 format.ts, period.ts, backup.ts
  components/  — общие UI-компоненты и иконки
  features/    — модальные формы (операции, список операций)
  routes/      — экраны приложения
  store/       — UI-состояние (Zustand)
```

## Дизайн

Тёмная тема с красными акцентами по заданной палитре
(`#0B090A`, `#161A1D`, `#660708`, `#A4161A`, `#BA181B`, `#E5383B`, серые и белый).

## Заметки

- Котировки акций вводятся вручную. Архитектура готова к подключению биржевого API
  (например, ISS API Мосбиржи) без переработки модели данных.
- Данные хранятся только на устройстве. Для переноса между устройствами используйте
  экспорт/импорт в разделе «Настройки». Синхронизация через сервер — возможное
  будущее развитие.
